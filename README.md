# 🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. Play against the computer, which makes a random choice each round. The game displays the winner, keeps track of scores, and allows you to reset the game at any time.

---

**Live Demo:**  https://rock-paper-scissor-eight-eta.vercel.app

## 📌 Features

* 🎮 Player vs Computer gameplay
* 🎲 Random computer choice every round
* ✊ Rock, 📄 Paper, ✂️ Scissors selection
* 🏆 Automatic winner detection
* 📊 Live score tracking
* 🔄 Reset game functionality
* 🎨 Modern and responsive user interface
* ⚡ Built with vanilla JavaScript (no frameworks)

## 🎯 How to Play

1. Open the game in your browser.
2. Click **Rock**, **Paper**, or **Scissors**.
3. The computer randomly selects its choice.
4. The game compares both choices and displays the result.
5. Scores are updated after each round.
6. Click **Reset Game** to start over.

---

## 🧠 Game Rules

* 🪨 Rock beats Scissors
* ✂️ Scissors beats Paper
* 📄 Paper beats Rock
* If both players choose the same option, the round is a draw.
